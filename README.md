# VR Inventory and Weapon System

![Unity](https://img.shields.io/badge/unity-%23000000.svg?style=for-the-badge&logo=unity&logoColor=white)
![C#](https://img.shields.io/badge/c%23-%23239120.svg?style=for-the-badge&logo=c-sharp&logoColor=white)
![VR](https://img.shields.io/badge/VR-XR%20Interaction%20Toolkit-blue?style=for-the-badge)

An advanced framework for **Virtual Reality** focused on modular inventory management and realistic weapon interaction. Developed in Unity using the **XR Interaction Toolkit**, this project demonstrates industry-standard patterns for immersive VR experiences.

---

## 🚀 Key Features

* **Modular Inventory System**: Efficient object management and storage logic designed specifically for spatial environments.
* **Advanced XR Interactions**: Implementation of grab, poke, and ray interactors for high-fidelity user input.
* **Weapon Mechanics**: Custom scripts for handling, holstering, and interacting with specialized VR equipment.
* **Scalable Architecture**: Clean project structure separating core VR logic from assets and project settings.

## 📁 Project Structure

* **`/Assets`**: Core 3D models, materials, and prefabs.
* **`/VR`**: Primary C# scripts and interaction logic.
* **`/Packages`**: Project dependencies including OpenXR and XR Interaction Toolkit.
* **`/ProjectSettings`**: Optimized VR configurations for performance and compatibility.

## 🛠️ Technical Requirements

* **Unity Version**: 2022.3+ (LTS recommended)
* **XR Plugin**: OpenXR / XR Interaction Toolkit
* **Target Platforms**: Meta Quest (2/3/Pro), Rift, and SteamVR compatible headsets.

---

## ⚙️ Setup & Installation

1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/Isabela-Tellez/InventarioArmasVR.git](https://github.com/Isabela-Tellez/InventarioArmasVR.git)
    ```
2.  **Open with Unity Hub**: Select the project folder and ensure you have the correct Unity version installed.
3.  **Build Settings**: Switch to your target platform (Android for Quest or PC Standalone).
